2.需要总结的知识点

示例中，有以下知识点需要总结

  1. Html+CSS+JS实现小目标，其中html负责主要的内容框架，Css负责添加静态的样式，Js负责增加动态效果。
  2. html部分，导航栏内容放在一个盒子里面，并且转为块级元素，进行宽高的设置，在代码中设置类名和时间类型；内容部分，分别用三个盒子进行表示，设置相应的类名和ID名。
  3. CSS部分，给导航栏添加样式，由于要设置点击变换后的样式，因此对于背景，设置了bg和changebg两个样式；内容部分，对内容进行隐藏，点击相应导航栏时，对内容进行调用。
  4. JS部分是最重要的一点，负责书写事件驱动程序。利用一个循环负责为三个导航块添加功能。首先，点击某一个导航块后，把所有的导航块的背景设置为bg,内容区进行隐藏，然后把调用该事件的导航块自己的背景色进行切换，其把对应的内容区设置为显示。
